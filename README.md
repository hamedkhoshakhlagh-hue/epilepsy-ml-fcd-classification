# Epilepsy Machine Learning Analysis Pipeline

This repository contains the code, dataset, and statistical analysis files used in the study investigating machine learning-based classification approaches in epilepsy neuroimaging data.

## Repository Contents

### 1. Jupyter Notebook
**File:** `Copy_of_EpilepsyFullJourney(v2).ipynb`

This notebook contains the complete machine learning workflow, including:

- Data preprocessing
- Feature selection
- Correlation filtering
- Principal Component Analysis (PCA)
- Model training and evaluation
- Cross-validation procedures
- ROC curve analysis
- Bootstrap confidence interval estimation
- Statistical comparisons between classifiers

---

### 2. Dataset
**File:** `df_destx.csv`

This file contains the dataset used for machine learning analyses.

---

### 3. SPSS Statistical Analysis File
**File:** `StatisticalAnalyses.spv`

This file contains the original SPSS statistical analysis output.

---

### 4. SPSS Output in Word Format
**File:** `OUTPUT.doc`

This document contains exported statistical analysis results from SPSS in Microsoft Word format.

---
